================================================================================
                         FOR NEW USERS CLONING
================================================================================

Quick Start (5 minutes):
  1. git clone https://github.com/Syfudeen/Skorly.git
  2. cd Skorly
  3. npm install && cd backend && npm install && cd ..
  4. echo "VITE_API_URL=http://localhost:5000" > .env
  5. cd backend && cp .env.example .env && cd ..
  6. docker-compose -f backend/docker-compose.yml up -d mongodb
  7. Terminal 1: cd backend && node src/server-working.js
  8. Terminal 2: npm run dev
  9. Open http://localhost:8080

Complete Guide:
  → Read: CLONE_AND_RUN.md (5 min read)
  → Follow: Step-by-step instructions
  → Reference: QUICK_REFERENCE_CARD.md if needed
